# Rates Analytics Engine

Practical fixed income pricing and risk toolkit for trading applications.

## Features
- Yield curve construction (bootstrapping)
- Bond pricing and cashflow modeling
- Interest rate swap valuation
- Risk measures (DV01, duration, sensitivities)

## Structure
- `src/` : Core pricing and risk logic
- `notebooks/` : Demonstrations and visualizations

## Objective
To implement core analytics used on rates trading desks in a clean and modular way.
